# AI-Powered Customer Subscription Analysis

## Project Overview
This project demonstrates an AI-powered analytics assistant designed to analyze a 5,000-record customer subscription dataset and answer business-focused questions. The AI agent allows users to interact with structured subscription data through an intuitive web interface, transforming analytical questions into clear, data-driven insights that support business analysis and decision-making.

The project combines data analytics fundamentals with an AI agent to make subscription data more accessible and easier to interpret, focusing on customer activity, subscription plans, churn behavior, pricing trends, and signup patterns.


---

## Tools & Technologies
- **Excel** – Original Dataset
- **MySQL** – Database creation and analytical querying.
- **AI Agent (Chatbase / ChatGPT)** – Interactive question-answering  
- **HTML** – Web-based user interface

  
---

## Dataset Description
The analysis is based on a customer subscription dataset containing the following fields:

- **Customer_ID** – Unique identifier for each customer  
- **Subscription_Category** – Type of service subscribed to (e.g., Music, Sports, Gaming, Education)  
- **Subscription_Plan** – Pricing tier selected (Basic, Standard, Premium)  
- **Monthly_Price** – Monthly subscription cost  
- **Signup_Date** – Date the customer signed up  
- **Status** – Current subscription status (Active or Churned)  

The dataset is used to evaluate customer activity, subscription performance, pricing trends, and signup behavior.

---

## AI Bot Capabilities
The AI assistant is designed to answer the following business-focused questions based on the dataset:

- How many total customers are there?
- How many customers are currently active?
- How many customers have churned?
- How many customers are subscribed to each subscription plan?
- Which subscription plan has the highest number of active customers?
- How many customers are subscribed to each subscription category?
- Which subscription category has the highest number of churned customers?
- How does the average monthly subscription price vary by month across all three years?
- Which subscription plan generates the highest estimated monthly revenue?
- How many customers signed up in each year?

---

## Key Insights
- The dataset contains a total of 5,000 customer subscription records.
- Out of all customers, 3,445 currently have active subscriptions, while 1,555 customers have churned.
- Customer distribution across subscription plans is balanced, with 1,691 customers on the Standard plan, 1,673 on the Premium plan, and 1,636 on the Basic plan.
- The Standard subscription plan has the highest number of active customers, totaling 1,161 active users.
- Customer subscriptions are fairly evenly distributed across categories, with Music having the highest number of subscribers (1,016) and Education the lowest (969).
- The Education subscription category has the highest number of churned customers.
- Average monthly subscription prices remain largely stable across months, with the highest values observed in August and September.
- Revenue analysis shows that the Premium subscription plan generates the highest estimated monthly revenue at $33,443.27.
- Customer signup trends indicate that new subscriptions were highest in 2022, followed by a gradual decline in 2023 and 2024.
